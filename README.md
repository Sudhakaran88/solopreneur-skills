Opinionated, research-backed marketing and growth skills for your AI coding agent. Built for one-person teams who need to punch above their weight — not enterprise marketing departments.

> If ChatGPT gives you "best practices," this repo gives you **what practitioners on Reddit actually say works in 2026** — with named frameworks, real benchmarks, and contrarian takes sourced from primary research.

---

## What you get

- **42 skills**, each a self-contained expert playbook (CRO, SEO, cold email, pricing, GTM, churn, paid ads, and more)
- **Opinionated** — every skill takes a stance. No hedging, no "it depends" soup.
- **Research-backed** — benchmarks with year + source; named experts (CXL, Demand Curve, Lenny's Newsletter, SparkToro, Princeton GEO paper, Unbounce, Klaviyo, Baymard)
- **Practitioner voice** — Reddit quotes and real founder experiences baked into every skill
- **Solopreneur-first** — written for one-person teams, not teams of 20
- **Cross-referenced** — every skill links to related skills so your agent always knows the next step
- **Under 500 lines each** — dense, not bloated

## Quick install — Claude Code

```bash
# Inside Claude Code
/plugin marketplace add Sudhakaran88/solopreneur-skills
/plugin install solopreneur-skills
```

Then just describe what you want:

```
"my landing page isn't converting"       → triggers page-cro
"write me a cold email sequence"          → triggers cold-email
"how should I price my SaaS?"             → triggers pricing-strategy
"optimize for ChatGPT and Perplexity"     → triggers ai-seo
```

Skills activate automatically based on intent. See [`docs/installation.md`](./docs/installation.md) for Cursor, Codex, Copilot, Gemini, and manual use.

## How skills work

Every skill is a single `SKILL.md` file with YAML frontmatter that tells Claude Code when to invoke it:

```
skills/
├── page-cro/
│   └── SKILL.md
├── cold-email/
│   └── SKILL.md
└── ...
```

When you ask Claude something like *"my signup conversion dropped this week"*, the agent reads the `description` fields across all skills, matches the intent, and loads the full `signup-flow-cro` SKILL.md — giving you a focused expert session instead of generic LLM advice.

Each skill includes:

- **Opinionated framework** (not a list of 47 tactics)
- **Benchmarks with sources + year** (Unbounce Q4 2024, Princeton GEO paper, etc.)
- **Common Mistakes** from Reddit/forums
- **Contrarian Takes** that push back on textbook advice
- **Related Skills** for what to do next

See [`docs/skill-creation-guide.md`](./docs/skill-creation-guide.md) for the full format spec.

## Why this exists

Most AI marketing advice is trained on 2019 blog posts written by SEO agencies trying to rank for "best X tips." It's generic, it hedges everything, and it hasn't caught up to 2024–2026 reality: AI Overviews, HCU, post-MPP email, Gmail bulk sender rules, scaled-content-abuse policy, the Andromeda algorithm.

This repo is the alternative: **primary-source-researched, opinionated, solopreneur-first.** Every skill is a condensed version of what you'd get from 20 hours of reading r/SaaS, r/marketing, CXL, Lenny's, Demand Curve, First Round, and the Princeton GEO paper — minus the theatre.

It's free and MIT licensed. Take it, fork it, adapt it. If you improve something, [PRs welcome](./CONTRIBUTING.md).
